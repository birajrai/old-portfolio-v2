Kira

A spigot plugin that helps staff with better quality of some life changes

  • Compatibility: 1.8.8
  • The commands are divided into two categories: Staff Commands and Player Commands.
  • Staff Commands require the operator (op) status and can be used by server staff to manage the server and its players. These commands include the ability to heal or feed a player, vanish or reveal oneself, equip Staff Items, and switch between all chat or staff-only chat.
  • Player Commands, on the other hand, can be used by all players on the server to interact with the game environment. One such command is the report command, which sends a message to all staff with information on the player being reported. This information includes the number of times the player has been reported, and staff can click on the message to teleport to the reported player.
  • In addition to commands, there are also Staff Items that can be used by staff members. These items include the Wand of Banning, which allows a staff member to ban a player by left-clicking on them, the Potion of Vanishing, which allows a staff member to vanish or reveal themselves by right-clicking, and the Clear Inventory item, which allows a staff member to clear their inventory by right-clicking.
  • Overall, these commands and items provide an array of options for staff members to manage the server and its players, while also allowing players to interact with the game world in unique ways.

BetterUhc

Automates UHC games on a dedicated 1.8 - 1.18 server.

  • To begin with, you will need a dedicated server that can support Minecraft gameplay. The server should have a strong CPU and plenty of RAM to handle the demands of running the game and the automation software. It is also important to choose a version of Minecraft that is compatible with the automation software and any plugins you plan to use. Versions 1.8 through 1.18 are currently the most popular for UHC games.
  • Next, you will need to install an automation plugin or custom program to manage the game rules and settings. Popular automation plugins for UHC games include Ultra Hardcore, UHC Core, and UHC Ultimate. These plugins allow you to customize the game world, distribute resources, and set other rules and conditions for gameplay.
  • Once the automation software is installed, you can create and manage UHC games through the plugin's interface or custom program. This may involve setting up game modes, creating teams, or customizing the shrinking of the game world. It is important to thoroughly test the automation software before hosting any games to ensure that everything runs smoothly and as intended.
  • Finally, you may want to consider promoting your UHC server to attract players. This can be done through social media or Minecraft server lists, and by creating engaging and fun gameplay experiences for your players.
  • Overall, setting up and running an automated UHC server requires careful planning and attention to detail, but with the right tools and approach, it can provide an exciting and engaging Minecraft gameplay experience for both players and server administrators.
